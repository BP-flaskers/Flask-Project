# 사용자 적성 추천 웹 프로젝트
---
## 1. 문제
많은 사람들이 자신에게 잘 맞는 직종에서 일하거나 자신의 역량에 대해서 100% 파악하지 못하는 경우가 많음. <br> 기존의 심리 테스트들은 사용자에게 단지 심리 분석에서만 끝나는 경우가 많음.
## 2. 해결책
선택지 기반 질문을 통해 사용자의 적성 및 선호 분야를 파악한다. <br>선택지 기반 추천 알고리즘을 사용한다. 
## 3. 대상
- 일반 사용자 : 자신의 적성을 알고 그에 걸맞는 분야를 추천 받을 수 있다.
- 상담 교육자/지도자 : 각 교육/지도자의 목적에 맞는 대상자들에게 웹 프로젝트에 맞는 결과를 이용해 상담 혹은 교육에 활용할 수 있다.
## 4. 주요기능
- 선택지 기반 추천 시스템 (적성, 선호분야)
- 테스트 결과 페이지 정보 제공
- RESTFUL API 설계
## 5. 경쟁/대안
| 플랫폼        | 특성                              |
|------------|---------------------------------|
| MBTI 테스트   | 성격유형 16가지로 분류                   |
| 커리어넷 진로심리검사 | 국내 공공기관에서 제공하는 심리기반 진로 적합도 평가   |
| 사람인 인적성검사  | 취업용 인적성 평가로 논리력, 추리력, 집중력 등을 점검 |
## 6. 기대효과
- 사용자가 적성을 명확하게 알고 그에 맞는 분야를 추천 받을 수 있다.
- 사용자가 자신의 심리테스트 결과를 알 수 있다.
- 상담/교육/지도자가 설문조사 내용을 바탕으로 각 목적에 알맞게 활용할 수 있다.
